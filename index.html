Sobre mí

Soy desarrollador de software con experiencia en aplicaciones móviles, desarrollo web, programas de escritorio, personalización de tiendas Shopify y proyectos con hardware. He trabajado con tecnologías como Flutter, Kotlin, Python, PyQt6, HTML, CSS, JavaScript, Liquid, Arduino UNO y ESP32.

Me gusta construir soluciones prácticas que combinen diseño, lógica de programación e integración con dispositivos físicos. He desarrollado apps móviles, interfaces web, programas en Python, soluciones para e-commerce y proyectos electrónicos aplicados a problemas reales.

También he realizado soluciones de integración para programas de terceros, conectando aplicaciones Android con software de escritorio mediante servidores locales, comunicación en red y puertos COM virtuales creados con com0com.

Mi objetivo es seguir creciendo como desarrollador, creando proyectos funcionales, ordenados y adaptados a las necesidades reales de los usuarios.

Integración Android-PC para Registro de Llamadas

Solución desarrollada en Kotlin para leer notificaciones entrantes de WhatsApp, extraer números telefónicos y enviarlos a un software de escritorio mediante un servidor local.

Para conectar la app con el programa de PC, utilicé puertos COM virtuales creados con com0com, permitiendo simular una entrada telefónica compatible con el registrador de llamadas del sistema.

Kotlin Android Servidor local com0com Puertos COM virtuales Automatización

Sistema Web de Gestión de Alquileres e Inventario

Sistema web desarrollado para administrar alquileres, inventario, usuarios, clientes, listas de precios e historial de movimientos. El proyecto permite organizar procesos internos, registrar pedidos, controlar materiales y mejorar la gestión de información de una empresa.

La solución fue construida con un backend en Django y Django REST Framework, junto a un frontend desarrollado con Vue.js y Vite. También incluye consumo de APIs, autenticación mediante token, manejo de rutas, servicios separados y componentes reutilizables.

Python Django Django REST Vue.js Vite JavaScript Axios JWT
